# Vrinda Store Sales Analysis Dashboard

## Overview

This project analyzes the 2022 sales data of Vrinda Store using Microsoft Excel. The objective is to transform raw transactional data into actionable business insights through an interactive dashboard.

The dashboard provides a comprehensive view of sales performance, customer demographics, order status, regional performance, and sales channels.

---

## Project Objectives

- Analyze annual sales performance.
- Identify top-performing states and sales channels.
- Understand customer demographics and purchasing behavior.
- Track monthly sales and order trends.
- Evaluate order fulfillment performance.
- Create an interactive dashboard for business decision-making.

---

## Key Business Insights

- Women customers generated the majority of sales.
- Adult customers represented the largest purchasing segment.
- Maharashtra emerged as the highest revenue-generating state.
- Online marketplaces contributed significantly to overall sales.
- Most orders were successfully delivered, indicating efficient operations.
